# Code of Conduct

This project expects a professional, technically focused tone in issues, pull requests, and review discussion.

## Expected Behavior

- Be respectful and direct.
- Critique code, claims, and evidence, not people.
- Keep reports factual and reproducible.
- Share only material that is safe to discuss in a public repository.

## Unacceptable Behavior

- harassment, threats, or personal attacks
- hateful or discriminatory language
- doxxing or posting private information
- deliberately posting secrets, private code, or other material that should not be public
- repeated bad-faith disruption after moderator direction

## Scope

This applies to repository spaces such as issues, pull requests, review comments, and other project discussion tied to this repo.

## Enforcement

Maintainers may edit, hide, lock, or remove content that violates this policy and may block repeat offenders from participating.

If you need to report conduct problems and cannot do it safely in public, use the contact options on the maintainer's GitHub profile if available. This repository does not promise a separate staffed moderation channel.
